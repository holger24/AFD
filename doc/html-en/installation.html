Installing AFD

If you install want to install from source, please read Compiling AFD. But it is easier to just download the RPM from here. To install the RPM you must ensure that you have the following packets installed:

  • RedHat/CentOS/Scientific Linux up to 6.x
      •       su -
      yum install openmotif libXaw libXpm xorg-x11-fonts-misc
  • RedHat/CentOS/Scientific Linux as of 7.x
      •       su -
      yum install motif libXaw libXpm xorg-x11-fonts-misc
  • Fedora up to Version 17
      •       su -
      yum install openmotif libXaw libXpm xorg-x11-fonts-misc
  • Fedora Version 18 - 21
      •       su -
      yum install motif libXaw libXpm xorg-x11-fonts-misc
  • Fedora as of Version 22
      •       su -
      dnf install motif libXaw libXpm xorg-x11-fonts-misc
  • Ubuntu
      •       sudo apt-get install libmotif
  • SLES up to 11
      •       su -
      zypper install openmotif
  • SLES as of 12 and openSUSE
      •       su -
      zypper install motif

To install the RPM, just enter the following command: 

           rpm -Uvh afd-1.4.5-4.el6.x86_64.rpm

If AFD is already installed and you just want to update AFD the command is as follows: 

           rpm -Fvh afd-1.4.5-4.el6.x86_64.rpm
